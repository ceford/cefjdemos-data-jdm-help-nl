## Beschrijving

Het *Ingelogde Gebruikers* module type toont een lijst van gebruikers die momenteel op de site zijn ingelogd. Het wordt weergegeven in het Startdashboard.

### Algemene Elementen

Sommige elementen van deze pagina worden behandeld in afzonderlijke Help-artikelen:

* [Werkbalken](jdocmanual?article=help/common-elements/toolbars).
* [De Modules: Module Tab](jdocmanual?article=help/modules/modules-module-tab).
* [De Modules: Geavanceerde Tab](jdocmanual?article=help/modules/modules-advanced-tab).
* [De Machtigingen Tab](jdocmanual?article=help/common-elements/edit-permissions).

## Hoe te Toegang

- Selecteer **Systeem → Beheer Paneel → Beheerdersmodulen** uit
  het Beheerdersmenu. Vervolgens...
  - Om een nieuwe module te maken: selecteer de knop **Nieuw** in de werkbalk. Vervolgens...
    - Selecteer het vereiste modulype.
  - Om een bestaande module te bewerken:
    - Zoek de module in de lijst van geïnstalleerde modules en selecteer de
      titel link in de kolom **Titel**.

## Formulier Velden

- **Titel** De titel van de module. Dit is ook de titel die wordt weergegeven
  voor de module, afhankelijk van het *Titel Weergeven* formulier veld.

### Module Tab

#### Linker Paneel

- **Gebruikers om weer te geven** Het aantal gebruikers dat moet worden weergegeven (standaard 5).
- **Naam** De naam of gebruikersnaam van de ingelogde gebruiker weergeven.
